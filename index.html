📋 FORMATOS DE JOGO
No Warhammer 40K, o tamanho da batalha define quantas unidades cada jogador pode usar. Use isso como referência para equilibrar os exércitos.
🎯 Combat Patrol
~500 pontos · 3 CP
Ideal para aprender o jogo. 2-4 unidades por lado. Partida de 45-60 min. Recomendado para iniciantes.
⚔ Incursão
~1000 pontos · 4 CP
Batalha intermediária. 5-8 unidades por lado. Partida de 90-120 min.
🔥 Strike Force
~2000 pontos · 6 CP
Batalha padrão de torneio. 10-15 unidades por lado. Partida de 3+ horas.
💥 Onslaught
~3000 pontos · 8 CP
Batalha épica. Para jogadores experientes com muito tempo disponível.
💡 DICA PARA SEU GRUPO
Combinem antes quantas unidades cada jogador pode usar. Para uma primeira partida, sugerimos 3-5 unidades por lado.
